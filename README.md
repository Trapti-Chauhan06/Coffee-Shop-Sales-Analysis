# Coffee Shop Sales Dashboard

# Overview
An interactive Power BI dashboard built to analyze coffee shop sales performance using KPIs, sales trends, customer behavior, and product performance.

# Objective
To create an interactive dashboard that provides meaningful business insights for better decision-making.

# Tools Used
- Power BI
- Microsoft Excel
- DAX

# Dataset
The dataset contains 1,000 coffee shop transactions with sales, profit, customer, product, and payment details.

# Dashboard Features
- Total Sales
- Total Profit
- Total Transactions
- Average Customer Rating
- Monthly Sales Trend
- Sales by Category
- Sales by Store Location
- Sales by City
- Top 10 Products
- Payment Method Distribution
- Customer Type Analysis
- Customer Rating Distribution

# Interactive Filters
- Transaction Date
- Store Location
- City
- Product Category
- Product Name
- Customer Type
- Payment Method

# Key Insights
- Coffee products lead in sales.
- University is the top-performing store.
- Walk-in customers spend more.
- UPI is the preferred payment method.
- Customer ratings remain high.
